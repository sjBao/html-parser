# HTML Parser
demo: https://vast-inlet-80619.herokuapp.com/

## running on your machine:
clone the repository

1. **Rails:**
* `$ bundle`
* `$ rails db:create`
* `$ rails db:migrate`


2. **npm:**
* `$ npm run build`

After you've finish the set up you can run on your machine with `$ rake start`.
